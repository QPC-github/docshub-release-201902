Struct std::alloc::System

pub struct System;

The default memory allocator provided by the operating system.

This is based on malloc on Unix platforms and HeapAlloc on Windows, plus related functions.

This type can be used in a static item with the #[global_allocator] attribute to force the global allocator to be the systemâ€™s one. (The default is jemalloc for executables, on some platforms.)

use std::alloc::System;

#[global_allocator]
static A: System = System;

fn main() {
    let a = Box::new(4); // Allocates from the system allocator.
    println!("{}", a);
}

It can also be used directly to allocate memory independently of the standard libraryâ€™s global allocator.
