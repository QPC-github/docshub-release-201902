KMutableProperty2

kotlin-stdlib / kotlin.reflect / KMutableProperty2 
interface KMutableProperty2<D, E, R> : 
    KProperty2<D, E, R>, 
    KMutableProperty<R>

Represents a var-property, operations on which take two receivers as parameters.

get

 
abstract fun get(receiver1: D, receiver2: E): R

getDelegate
