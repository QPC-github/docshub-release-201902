The qtmain Library

qtmain is a helper library that enables the developer to write a cross-platform main() function on Windows. If you do not use qmake or other build tools such as CMake, then you need to link against the qtmain library.

The qtmain library is statically linked with the application. To avoid issues with static linking and the GNU Lesser General Public License (LGPL), it is licensed both under the commercial licenses, and the BSD 3-clause "New" or "Revised" License.
