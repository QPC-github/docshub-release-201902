C++ named requirements: ContiguousContainer

A ContiguousContainer is a Container that stores objects in contiguous memory locations.

Requirements

The type X satisfies ContiguousContainer if.

  • The type X satisfies Container
  • The type X supports RandomAccessIterators
  • The member types X::iterator and X::const_iterator are ContiguousIterators

Contiguous containers in the standard library

basic_string
 stores and manipulates sequences of characters
(class template)
array
(C++11)
 static contiguous array
(class template)
vector
 dynamic contiguous array
(class template)
