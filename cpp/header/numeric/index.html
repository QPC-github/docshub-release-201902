Standard library header <numeric>

This header is part of the numeric library.

Functions

iota
(C++11)
 fills a range with successive increments of the starting value
(function template)
accumulate
 sums up a range of elements
(function template)
reduce
(C++17)
 similar to std::accumulate, except out of order
(function template)
transform_reduce
(C++17)
 applies a functor, then reduces out of order
(function template)
inner_product
 computes the inner product of two ranges of elements
(function template)
adjacent_difference
 computes the differences between adjacent elements in a range
(function template)
partial_sum
 computes the partial sum of a range of elements
(function template)
inclusive_scan
(C++17)
 similar to std::partial_sum, includes the ith input element in the ith sum
(function template)
exclusive_scan
(C++17)
 similar to std::partial_sum, excludes the ith input element from the ith sum
(function template)
transform_inclusive_scan
(C++17)
 applies a functor, then calculates inclusive scan
(function template)

(C++17)
 applies a functor, then calculates exclusive scan
