aws_rds - rds instance source

  • Synopsis
    • Requirements
  • Parameters
  • Examples
  • Status
    • Author

Synopsis

  • Get instances and clusters from Amazon Web Services RDS.
  • Uses a <name>.aws_rds.yaml (or <name>.aws_rds.yml) YAML configuration file.

Requirements

The below requirements are needed on the local master node that executes this inventory.

  • boto3
  • botocore

Examples

plugin: aws_rds
regions:
  - us-east-1
  - ca-central-1
keyed_groups:
  - key: 'db_parameter_groups|json_query("[].db_parameter_group_name")'
    prefix: rds_parameter_group
  - key: engine
    prefix: rds
  - key: tags
  - key: region
