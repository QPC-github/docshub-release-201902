GDScript

Inherits: Script < Resource < Reference < Object

Category: Core

Brief Description

A script implemented in the GDScript programming language.

Member Functions

PoolByteArray get_as_byte_code ( ) const
Object new ( ) vararg

Description

A script implemented in the GDScript programming language. The script exends the functionality of all objects that instance it.

new creates a new instance of the script. Object.set_script extends an existing object, if that object’s class matches one of the script’s base classes.

Member Function Description

PoolByteArray get_as_byte_code ( ) const

Returns byte code for the script source code.

Object new ( ) vararg

Returns a new instance of the script.

For example:

var MyClass = load("myclass.gd")
var instance = MyClass.new()
assert(instance.get_script() == MyClass)
