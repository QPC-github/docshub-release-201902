push

The push event is fired to indicate that a Service Worker has received a push message.

General info

Specification
Push API
Interface
PushEvent
Bubbles
No
Cancelable
No
Target
Service worker
Default Action
None

Properties

data
Provides access to the data payload of a push message, contained in a PushMessageData object.

Examples

self.addEventListener('push', function(event) {
  var obj = event.data.json();
  port.postMessage(obj);
});
